# NouriWamp

Ce sous-dépôt du projet NouriWamp contient une copie de tous les binaires des modules, inchangés depuis leur emplacement d'origine.

Ils sont hébergés sur GitHub afin d'éviter les liens brisés.

Les fichiers binaires non disponibles sous forme d'archives ou de fichiers individuels sont recompilés pour NouriWamp, mais les fichiers originaux sont également inclus.
